Privacy Policy — Manga Tint

Last updated: July 12, 2026

Manga Tint was formerly published as Colored Glasses (色めがね). This policy applies to the extension under both names.

1. Data Collection

The extension does not collect, transmit, or store any personal or sensitive user data. All processing occurs locally in the browser, without external communication.

2. Local Data

The extension does not persistently save user settings or data. It does not access or store page content.

3. Purpose of Use

Since no data is collected, there is no use for analytics, advertising, or tracking.

4. Sharing and Third Parties

No data is shared with third parties. No SDKs, analytics tools, or ads are included.

5. Browser Permissions

  • activeTab: Apply temporary color filters to the current page upon user action
  • scripting: Insert and execute the filter script

No browsing history, network access, or persistent storage permissions are requested.

6. Retention and Deletion

No persistent data is stored. Removing the extension (uninstalling) fully disables it.

7. Children's Privacy

The extension is not directed at children under 13 and is not designed to collect their information.

8. Security

Because no external transmission occurs, risk of data leakage is minimized.

9. Changes

We may update this policy. Any significant changes will be posted here with an updated "Last updated" date.

10. Contact
