# Colored glasses

View the web with colored glasses. It is possible to see the plaster statue in gold, silver, copper, and bronze.

## Building and running on localhost

First install dependencies:

```sh
yarn install
```

To create a production build:

```sh
yarn run build
```

Running

```sh
yarn run dev
```
