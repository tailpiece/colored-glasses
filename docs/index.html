ブラウザ拡張機能（PC用）

色めがね

見ているページ全体を、色眼鏡を通して金や銀、chocolate mintに染めます。普段と異なる見た目を求める方に。

このアドオンを使う
How to use

使い方

ページを表示する

色眼鏡を通したいページにアクセスします。
※全てのページに対応しているわけではありません。

好きなパターンを選ぶ

機能拡張をクリックすると、カラーリストが表示されるので、好きな色を選んでクリックしてください。

完了

ページが色眼鏡を通して確認できます。

Precautions

注意事項

この機能を使う前にご確認ください

禁止事項

「利用者自身が著作権を持っていない」コンテンツに対し、「本機能拡張を適用した状態」の「スクリーンショットや、画面の一部が映り込んだ状態」の画像や動画を、公開・配布・放映することを禁止します。 （例：「他人の作品に色付けた状態が含まれているスクショ・動画」の公開禁止。自分のだったらOK）

※このページは、色めがね適用した状態でスクショOKです

対応できない不具合について
  • 適用後のページが、意図しない表示（動作）をすることがあります。これはcss filterの仕様なので、ご理解・ご了承ください（例えばYahooのスクロール追従メニューの挙動がおかしくなったり、twitterのポップアップが出てこなくなったり）。
  • 上記の現象は、適用する前の状態に、デベロッパーツールでbodyタグに「 filter: blur(2px); 」を付与すると、再現できます。
免責事項

  • 本機能は、予告なく仕様を変更することがあります。

Supplementary information

補足事項

グラデーションマップを使っています。より具体的には、CSSのSVG filterをgradientmaps.jsで追加しています。
この拡張機能は「アクセスしているページに対して、グラデーションマップをかける」ためです。SVGフィルタを動的にHTMLに書き加えた後、指定箇所のCSSを編集するため、この許可が必要になります。
画像によりますが、基本的に白の個所は白いままです。このため、グレースケールで構成されている画像に適していて、影絵などには適していません。また、最初からカラーで塗られている画像には　ほぼ逆効果です。
この機能拡張の仕組み上、不可能です。ON/OFF以外は諦めてください。
注意事項にも記載していますが、CSS filterの仕様です。この機能拡張は、機能拡張をクリックした時に表示される画面の「ポップアップ側の機能」と、 表示中のページに対して働きかける「コンテンツ側の機能」の２つに分れているのですが、主に下記の処理を行っています。

（1）ポップアップ側の機能
・グラデーションを選び、画面への反映
・グラデーションマップの解除
・任意のグラデーション（１種類）の画面への反映、編集・リセット・保存

（2）コンテンツ側の機能（表示しているページに対しての処理）
・(1)で指定されたカラーを、グラデーションマップで適用または解除する


  • パレットアイコンをクリックすることで、グラデーション編集機能に変わる
  • 色を適用（ローラーアイコンをクリックするか、プリセットの色をクリック）するまでは、グラデーションを編集しても反映しない
  • 適用した後は、（ポップアップを閉じるまで）リアルタイムに反映するようになる
  • 消しゴムアイコンは、グラデーションマップの解除
  • ゴミ箱アイコンは、編集したグラデーションマップを初期状態に戻す（保存してあった場合、グラデーションマップ情報も消す）
  • １色のみ保存可能
  • 保存するまでは、保存しない。
  • ポップアップを閉じると、保存していないグラデーション情報は失われる。
  • 保存アイコンのボタン色が緑になった時は、保存しているグラデーションと違うグラデーションになっている。保存の押し忘れに注意
  • 保存したグラデーションは、プリセットの一番右下に表示されるようになる。
  • 保存したグラデーションは、ポップアップを開きなおすと、編集画面に反映される。
  • プリセットを選ぶと、プリセットのカラー情報がエディタに反映される（反映されるだけで、保存を押すまで保存されない）
※ちなみに、すみません。firefoxは現状　編集画面では、グラデーションの比率を変更できても、グラデーションの色を編集することは出来ないようです。
編集ファイルをお持ちの場合、編集ファイルで直接グラデーションマップを適用していただいたほうが、綺麗な結果が得られるかと思います。
この機能拡張は主に「１つのグラデーションマップで　それっぽく見せる」のが目的なので、特定の箇所の色を変更するのは難しいですが、ご自身で選択範囲を指定いただいて、その個所にグラデーションマップをかけることで、素敵な作品に出来上がることでしょう。

殆どの制作アプリケーションは、グラデーションマップ機能に対応しています。
Adobe Photoshop, CLIP STUDIO, procreate, ibisPaint, MediBang Paint, etc...
